🔍 Mirror API

请求镜子 - 看清你的每一次HTTP请求

📖 简介

Mirror API 是一个用于调试和测试的API服务，它可以返回 incoming 请求的详细信息，就像照镜子一样让你看清请求的原貌。

无论你是前端开发者、API测试工程师还是学习者，这个工具都能帮助你更好地理解和调试HTTP请求。

版本: 1.0.0 | 服务器时间: {{ timestamp }}

✨ 核心特性

  • 支持所有HTTP方法：GET, POST, PUT, DELETE, PATCH, OPTIONS, HEAD
  • 实时显示请求头信息、查询参数、请求体数据
  • 自动解析JSON和表单数据
  • 显示客户端信息（IP地址、User-Agent等）
  • 提供简洁版和详细版两种响应格式

🌐 API 端点

/mirror

简洁版镜像 - 返回核心请求信息，适合快速调试

GET POST PUT DELETE PATCH OPTIONS HEAD
curl "http://localhost:5000/mirror?name=test&age=25"
curl -X POST http://localhost:5000/mirror -H "Content-Type: application/json" -d '{"test": "data"}'

/mirror/detail

详细版镜像 - 返回完整的请求详细信息，适合深度调试

GET POST PUT DELETE PATCH OPTIONS HEAD
curl "http://localhost:5000/mirror/detail?param1=value1&param2=value2"
curl -X POST http://localhost:5000/mirror/detail -H "Authorization: Bearer token" -d "username=test&password=secret"

/health

健康检查 - 验证API服务是否正常运行

GET
curl http://localhost:5000/health

/info

API信息 - 显示JSON格式的API信息

GET
curl http://localhost:5000/info

🚀 快速开始

  1. 启动服务器: python app.py
  2. 测试健康检查: curl http://localhost:5000/health
  3. 尝试简洁版: curl "http://localhost:5000/mirror?test=hello"
  4. 尝试详细版: curl -X POST http://localhost:5000/mirror/detail -H "Content-Type: application/json" -d '{"data": "test"}'
  5. 查看完整文档: 直接在浏览器访问 http://localhost:5000/

📋 请求示例

GET带参数
curl "http://localhost:5000/mirror?page=1&limit=20"
POST JSON
curl -X POST http://localhost:5000/mirror -H "Content-Type: application/json" -d '{"username": "test"}'
POST 表单
curl -X POST http://localhost:5000/mirror -H "Content-Type: application/x-www-form-urlencoded" -d "name=John&age=30"
带认证头
curl -X GET http://localhost:5000/mirror -H "Authorization: Bearer token123"
